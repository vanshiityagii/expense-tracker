# Expense Tracker Web App

A full-stack web application built with Python and Flask that helps users track their daily expenses with data visualization.

## Live Demo
🌐 [Click here to view live app](https://expense-tracker-redc.onrender.com)

## Features
- User Registration and Login with secure password hashing
- Add and Delete expenses
- Categorize expenses (Food, Travel, Bills, Shopping, Entertainment)
- Pie chart showing spending by category
- Total expense summary dashboard
- Each user sees only their own expenses

## Tech Stack
- **Backend:** Python, Flask, Flask-Login, Flask-SQLAlchemy
- **Database:** SQLite
- **Frontend:** HTML, Bootstrap 5, Jinja2
- **Charts:** Chart.js
- **Deployment:** Render.com, Gunicorn
- **Version Control:** Git, GitHub
